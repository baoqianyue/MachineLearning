## [矩阵性质与应用]    
* 单位矩阵   

    **单位矩阵一定是方阵**   
    **$A^{0} = I$**  
    
* 矩阵的逆   

    矩阵中$AB = BA = I$,则称B是A的逆矩阵，记做：$B=A^{-1}$   
    A称为可逆矩阵(大多数)，或者叫非奇异矩阵(non-singular)   
    **可逆矩阵一定是方阵**   
    有些矩阵是不可逆的，称为奇异矩阵(singular)
